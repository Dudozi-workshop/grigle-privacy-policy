# 개인정보처리방침 – 그리글 (Grigle)

최종 업데이트일: 2026년 2월 16일

그리글(이하 “앱”)은 사용자의 개인정보 보호를 중요하게 생각합니다.

## 1. 수집하는 정보
그리글은 사용자의 개인정보를 수집하지 않습니다.

- 키보드를 통해 입력되는 텍스트는 기기 내에서만 처리됩니다.
- 입력 데이터는 저장되거나 외부 서버로 전송되지 않습니다.
- 이름, 이메일, 연락처, 위치 정보 등 개인식별정보를 수집하지 않습니다.

## 2. 권한 사용 목적
앱은 키보드 기능 제공을 위해 안드로이드 입력 권한을 사용합니다.  
입력 내용은 저장되거나 외부로 전송되지 않습니다.

## 3. 기기 내부 저장 정보
앱은 키보드 설정값 등 사용자 환경 설정을 기기 내부 저장소에 보관할 수 있습니다.

## 4. 제3자 제공
앱은 어떠한 개인정보도 제3자에게 제공하지 않습니다.

## 5. 광고 및 외부 서비스
현재 앱은 광고 및 외부 분석 도구를 사용하지 않습니다.

## 6. 향후 기능 추가
추후 기능이 추가될 경우 본 방침을 업데이트합니다.
